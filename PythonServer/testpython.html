    打开终端  执行python

    stm-macmini:~ apple$ python
    Python 2.7.10 (default, Jul 14 2015, 19:46:27)

    可以见到这样的东东

    开始建立我们的本地服务器了

    第一步:建立一个文件夹, 放在桌面或者其它地方都可以

    我在桌面上建立一个Service的文件夹(名字随便取)

    执行命令  cd  cd /Users/Apple/Desktop/Service  (转到Service 文件夹)

    第二步:开始Http服务器 :

     执行python -m SimpleHTTPServer 9999(这个不写的话默认端口是8000) 

    会显示下面的信息

    Service Apple$ python -m SimpleHTTPServer 9999
    Serving HTTP on 0.0.0.0 port 9999 ...

    如果提示  [Errno 48] Address already in use  表示 端口被占用 修改端口号就可以了 

    当然服务器开起了 这个窗口不能关


    第三步:测试

    拷贝一个Html文件到Service文件夹下面(test.html)

    打开浏览器firefox

    输入: localhost:9999/test.html
